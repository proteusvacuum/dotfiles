# dotfiles

Forked from https://github.com/holman/dotfiles

## install

```sh
git clone https://github.com/proteusvacuum/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```

## homebrew packages on macos

```sh
brew install cmake fzf gnu-tar htop libvterm ripgrep tmux

brew install --cask bitwarden claude-code emacs-app firefox firefox@developer-edition font-fira-code-nerd-font font-ibm-plex-mono font-maple-mono-nf ghostty
```
